# Nuvé — Brand Identity Kit

Brand identity guidelines for **Nuvé**, an intelligent wealth management platform by Acumen Holding (FRA Licensed, Egypt).

## Contents

- `index.html` — Full interactive brand kit covering:
  - Logo & Wordmark (4 lockups)
  - Clear Space Rules
  - Color Palette (6 colors)
  - Typography System (3 fonts)
  - Type Scale
  - Spacing System (8px grid)
  - UI Components (buttons, inputs, tags, radii)
  - Brand Voice (do/don't + personality pillars)
  - Imagery & Gradient Direction

## Brand Colors

| Name     | Hex       | Role                |
|----------|-----------|---------------------|
| Midnight | `#0A1628` | Primary Background  |
| Navy     | `#1A2942` | Cards & Surfaces    |
| Teal     | `#2EC4B6` | Primary Accent/CTA  |
| Gold     | `#D4A843` | Premium Accent      |
| Cream    | `#F5F0E8` | Light Background    |
| Slate    | `#8A95A5` | Body Text / Muted   |

## Typography

- **Display**: Cormorant Garamond (serif)
- **Body**: DM Sans (sans-serif)
- **Data/Labels**: Space Mono (monospace)

## Usage

Open `index.html` in any browser to view the full brand kit.
